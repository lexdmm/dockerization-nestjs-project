## Description

Dockerization of a NestJS typescript project

## Installation

```bash
$ npm install
```
## Docker compose to runing the postgres database

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
